AMS-Compose Documentation
==========================

**AMS-Compose** is a dependency management tool designed specifically for analog/mixed-signal IC design workflows. It provides selective library extraction, supply chain management, and intelligent filtering for analog design IP.

.. warning::
   **Important Disclaimer**: ams-compose is a dependency management tool. Repository owners are solely responsible for meeting all license compliance requirements with the IP checked into their repositories.
